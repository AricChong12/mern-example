# mern-example


## Tutorials By MongoDB University

Credits: https://www.mongodb.com/resources/languages/mern-stack-tutorial


This learning project showcases building a full-stack application using the MERN stack, combining MongoDB, Express.js, React, and Node.js. It includes CRUD operations—fetching, inserting, updating, and deleting data through API routes. I also made small modifications and added comments throughout the code to better understand how requests are handled, how data flows between frontend and backend, and how the database is managed, reinforcing practical learning through experimentation.


⚙️ ReactJS, ExpressJS, MongoDB, NodeJS
